Network Protocols

TCP/IP
(Transmission Control Protocol/Internet Protocol) These enable communication between computers on a network. TCP ensures data is delivered accurately and in sequence, while IP handless addressing and routing of packets.
HTTP
(Hypertext Transfer Protocol) This facilitates communication between web browsers and servers. it is used to access websites, submit forms, load images and videos. It powers the World Wide Web and is essential for web-based applications.
FTP
(File Transfer Protocol) This transfers files between computers over a network. It allows users to upload, download and manage files on remote servers.
